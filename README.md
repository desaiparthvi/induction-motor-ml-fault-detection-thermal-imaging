# Machine Learning Based Fault Detection in Induction Motors using Thermal Images

## Overview
This project focuses on developing machine learning models to detect faults in induction motors using thermal image data.

## Objectives
- Extract meaningful features from thermal images
- Train classification models
- Detect abnormal operating conditions

## Tools Used
Python, NumPy, Pandas, Scikit-learn, Matplotlib

## Status
Work in progress. Research paper under preparation.
